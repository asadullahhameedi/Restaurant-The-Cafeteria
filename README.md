## ✨ Features

### 🎨 Design & UI/UX
- ✅ Luxury dark theme with golden accents (`#c9a227`)
- ✅ Circular "AH" logo with animated gold border glow
- ✅ Glassmorphism sections with `backdrop-filter: blur()`
- ✅ Smooth hover animations & mouse-tracking glow effects
- ✅ Elegant typography: Playfair Display + Inter
- ✅ Fully responsive (mobile, tablet, desktop)

### 🎭 Interactive Animations
- ✅ Scroll-triggered reveal animations (fade/slide/scale)
- ✅ Animated statistics counter with ease-out cubic
- ✅ Floating reservation button with pulse animation
- ✅ Interactive menu cards with hover lift & zoom
- ✅ Gallery lightbox with zoom-in transition
- ✅ Auto-rotating testimonial slider with dot navigation
- ✅ Loading screen with dual-ring spinner animation

### ⚙️ Functional Components
- ✅ Sticky transparent navbar → glass on scroll
- ✅ Mobile hamburger menu with smooth slide-in
- ✅ Interactive menu filtering (All/Starters/Mains/Desserts)
- ✅ Reservation form with floating labels & real-time validation
- ✅ Dark/Light mode toggle with `localStorage` persistence
- ✅ Scroll progress indicator with gold gradient
- ✅ Social media links with hover glow effects

### ♿ Accessibility & Performance
- ✅ Semantic HTML5 structure for SEO
- ✅ ARIA labels & keyboard navigation support
- ✅ `prefers-reduced-motion` media query support
- ✅ Lazy-loaded images for faster performance
- ✅ GPU-accelerated CSS animations
- ✅ Zero external dependencies (pure HTML/CSS/JS)
